Critical Effect Template (Object Builder)
=========================================

Purpose:
- Visual effect for critical hit.
- Server constant already reserved: CONST_ME_CRITICAL_DAMAGE = 173.

Client object setup:
- Category: Effect / Magic Effect
- Effect ID: 173
- Name: critical_damage
- Width x Height: 32 x 32
- Pattern X: 1
- Pattern Y: 1
- Pattern Z: 1
- Layers: 1
- Frames: 6
- Loop: yes
- Approx frame duration: 60 ms

Visual style (frame-by-frame):
1. Small bright red core + tiny white spark in center.
2. Red starburst expands diagonally (4 short spikes).
3. Strong white-yellow center flash + red ring.
4. Ring grows, spikes fade, add 2-3 red shards.
5. Dark red smoke fragments around center.
6. Quick fade-out (mostly transparent red particles).

Color palette suggestion:
- #FFFFFF (small center highlight only)
- #FFD580 (brief warm flash)
- #FF3B30 (main red)
- #D80F0F (dark red)
- #5A0000 (fade particles)

Hook in server:
- Combat sends this effect on critical hits:
  - src/combat.cpp -> sendCriticalHitVisuals(...)
- Also sends floating text:
  - "CRITICAL!" in TEXTCOLOR_RED

Notes:
- If effect 173 is missing in client dat/spr, client will not display it correctly.
- Keep transparent background (no black fill) in all frames.
